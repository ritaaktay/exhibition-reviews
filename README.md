## Exhibition Reveiws

A web app for crowd-sourced exhibition reviews and realted discussions.

Users can add, browse and comment on exhibition reveiws.

## Demo

[https://exhibition-reviews.herokuapp.com/](https://exhibition-reviews.herokuapp.com/)

## Technologies
- MongoDB 5.0.9
- Express.js 4.18.1
- EJS 3.1.8
